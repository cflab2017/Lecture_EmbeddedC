# 과제 - 08. GPIO 입력 — 버튼/스위치 읽기

문제 1은 **C51(AT89C52, μVision 시뮬레이터)**, 문제 2는 **순수 로직(PC `gcc` 검증)** 입니다.

---

## 문제 1 — 버튼으로 LED 8개 제어
- 파일명: `homework_01/main.c`
- 핵심 개념: 입력 핀 준비(1 쓰기), 분기, 포트 전체 쓰기
- 플랫폼: C51 (AT89C52)

### 요구사항
- P3.3에 연결된 버튼(액티브 로우)을 읽는다.
- 버튼이 눌리면(0) P1의 LED 8개를 **모두 ON**(액티브 로우 → `P1 = 0x00`), 뗌(1)이면 **모두 OFF**(`P1 = 0xFF`).

### 예상 동작 (시뮬레이터)
- `Port 3`에서 P3.3을 0으로 만들면 `Port 1`이 0x00(LED 모두 ON), 1로 두면 0xFF(모두 OFF).

### 힌트
- 읽기 전에 `BTN = 1;`로 입력 준비.
- `sbit BTN = P3 ^ 3;`을 전역에 선언.

## 문제 2 — 버튼 누름 횟수 세기 (PC)
- 파일명: `homework_02/main.c`
- 핵심 개념: 에지 검출(1→0), 카운팅
- 플랫폼: 공통 (PC `gcc`)

### 요구사항
- 버튼 상태 시퀀스 `{1, 1, 0, 1, 0, 0, 1, 0}`(시간순, 1=뗌/0=눌림)에서 **누른 횟수**(하강 에지 1→0의 개수)를 센다.
- `presses = N` 한 줄을 출력한다. 형식은 "예상 출력"과 글자 단위로 일치해야 한다.

### 예상 출력
```
presses = 3
```

### 힌트
- `btn==0`인 횟수가 아니라 `prev==1 && cur==0`인 순간만 센다(길게 눌러도 1번).
- 직전 상태 `prev`를 1로 초기화하고 매 스텝 갱신한다.
