## 문제 1 — 포트 전체 쓰기 + sbit 비트 보정
- 파일명: `homework_01/main.c`
- 핵심 개념: SFR 전체 쓰기, `sbit` 개별 비트
- 플랫폼: C51 (AT89C52)

### 요구사항
- `P2`에 `0xAA`를 쓴 뒤, `sbit`으로 **P2.0을 1**, **P2.7을 0**으로 보정한다.
- 이후 `while(1)`로 값을 유지한다.

### 예상 동작 (시뮬레이터)
- `Peripherals → I/O-Ports → Port 2`에서 최종 **P2 = 0x2B** (0010 1011).
- 과정: `0xAA`(1010 1010) → P2.0 set(1010 1011=0xAB) → P2.7 clear(0010 1011=0x2B).

### 힌트
- `sbit P2_0 = P2 ^ 0;`, `sbit P2_7 = P2 ^ 7;`을 **파일 범위(전역)** 에 선언.
- 개별 비트 쓰기는 나머지 비트를 보존한다.

## 문제 2 — 포트 값 단계별 검산 (PC)
- 파일명: `homework_02/main.c`
- 핵심 개념: 비트 조작 누적 결과
- 플랫폼: 공통 (PC `gcc`)

### 요구사항
- `unsigned char p`를 `0x00`에서 시작해 아래를 순서대로 적용하고 매 단계 값을 `0x%02X`로 출력한다.
  1. 상위 니블 set (`p |= 0xF0`)
  2. bit4 clear
  3. bit0 toggle
- 출력 형식은 "예상 출력"과 글자 단위로 일치해야 한다.

### 예상 출력
```
step1 (set high nibble): 0xF0
step2 (clear bit4)     : 0xE0
step3 (toggle bit0)    : 0xE1
```

### 힌트
- 각 단계는 직전 결과 위에 누적된다.
- `0xF0`에서 bit4(0x10)를 clear하면 `0xE0`, bit0을 toggle하면 `0xE1`.
