# 과제 - 10. 외부·타이머 인터럽트

문제 1은 **C51(AT89C52, μVision 시뮬레이터)**, 문제 2는 **순수 로직(PC `gcc` 검증)** 입니다.

---

## 문제 1 — 타이머0 인터럽트로 LED 점멸
- 파일명: `homework_01/main.c`
- 핵심 개념: 타이머 인터럽트(interrupt 1), ISR 재리로드, IE 설정
- 플랫폼: C51 (AT89C52)

### 요구사항
- Timer0 Mode1으로 50ms마다 오버플로 인터럽트가 나도록 설정한다.
- ISR(`interrupt 1`)에서 TH0/TL0를 재리로드하고 P1.0 LED를 토글한다.
- `main`은 인터럽트만 켜고 빈 루프로 둔다.

### 예상 동작 (시뮬레이터)
- P1.0 LED가 약 50ms 간격으로 토글된다. 폴링 없이 인터럽트만으로 동작.

### 힌트
- `void timer0_isr(void) interrupt 1 { TH0=0x3C; TL0=0xB0; LED=!LED; }`.
- main에서 `TMOD`, `TH0/TL0`, `ET0=1`, `EA=1`, `TR0=1` 순으로 설정.
- Mode1은 자동 리로드가 아니므로 ISR에서 매번 재리로드.

## 문제 2 — 오버플로를 세어 0.5초마다 토글 (PC)
- 파일명: `homework_02/main.c`
- 핵심 개념: 인터럽트 카운트 누적, 분주
- 플랫폼: 공통 (PC `gcc`)

### 요구사항
- 오버플로 1회 = 50ms로 가정한다. **10회마다(=500ms)** LED를 토글한다.
- 1~30회 중 토글이 일어나는 시점에서 `overflow N (M ms): LED -> ON/OFF`를 출력한다.
- LED는 OFF에서 시작한다. 출력 형식은 "예상 출력"과 글자 단위로 일치해야 한다.

### 예상 출력
```
overflow 10 ( 500 ms): LED -> ON
overflow 20 (1000 ms): LED -> OFF
overflow 30 (1500 ms): LED -> ON
```

### 힌트
- `i % 10 == 0`일 때만 토글하고 출력한다.
- 경과 시간(ms) = `i * 50`. 형식은 `%2d`, `%4d`로 정렬.
