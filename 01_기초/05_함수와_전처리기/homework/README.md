## 문제 1 — 비트 순서 뒤집기 함수
- 파일명: `homework_01/main.c`
- 핵심 개념: 함수 분리, 비트 추출/조립
- 플랫폼: 공통

### 요구사항
- `uint8_t reverse_bits(uint8_t v)` 함수를 만들어 8비트의 순서를 뒤집는다(bit i → bit 7-i).
- 값 `0x01, 0x0F, 0x12, 0x80`에 대해 결과를 출력한다.
- 출력 형식 `reverse(0xNN) = 0xMM`은 "예상 출력"과 글자 단위로 일치해야 한다.

### 예상 출력
```
reverse(0x01) = 0x80
reverse(0x0F) = 0xF0
reverse(0x12) = 0x48
reverse(0x80) = 0x01
```

### 힌트
- `v`의 i번 비트가 1이면 결과의 `7 - i`번 비트를 set 한다.
- 결과 변수는 0으로 초기화하고 누적한다.
- 검산: `0x12 = 00010010` → 뒤집으면 `01001000 = 0x48`.

## 문제 2 — CLAMP 매크로
- 파일명: `homework_02/main.c`
- 핵심 개념: 함수형 매크로, 괄호 규칙
- 플랫폼: 공통

### 요구사항
- `CLAMP(x, lo, hi)` 매크로를 정의한다: `x`가 `lo`보다 작으면 `lo`, `hi`보다 크면 `hi`, 아니면 `x`.
- `CLAMP(5,0,10)`, `CLAMP(15,0,10)`, `CLAMP(-2,0,10)`을 출력한다.
- 출력 형식은 "예상 출력"과 글자 단위로 일치해야 한다.

### 예상 출력
```
CLAMP(5,0,10)  = 5
CLAMP(15,0,10) = 10
CLAMP(-2,0,10) = 0
```

### 힌트
- 삼항 연산자를 중첩한다: `((x) < (lo) ? (lo) : ((x) > (hi) ? (hi) : (x)))`.
- 인자와 전체 식을 모두 괄호로 감싼다(괄호를 빠뜨리면 호출부에서 깨진다).
